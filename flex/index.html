свойства флекс-контейнера

display: flex;flex | inline-flex

flex-direction - направление главной оси

flex-direction: row;слева направо (по умолчанию)

flex-direction: row-reverse;справа налево

flex-direction: column;сверху вниз

flex-direction: column-reverse;снизу вверх

justify-content - распределение флекс-элементов вдоль главной оси

justify-content: flex-start;в начале оси (по умолчанию)

justify-content: flex-end;в конце оси

justify-content: center;по центру

justify-content: space-around;распределяются равномерно вдоль всей оси

justify-content: space-between;равномерно (крайние элементы прибиваются к краям)

align-items - распределение флекс-элементов внутри строки вдоль поперечной оси

align-items: stretch;растягиваются по всей поперечной оси (по умолчанию)

align-items: flex-start;в начале оси

align-items: flex-end;в конце оси

align-items: center;по центру оси

align-items: baseline;по базовой линии

flex-wrap - перенос флекс-элементов на новую строку,
а также направление поперечной оси!

flex-wrap: nowrap;не переносятся (по умолчанию)

flex-wrap: wrap;переносятся (когда не помещаются на одной строке)

flex-wrap: wrap-reverse;переносятся, а также меняется направление

поперечной оси на противоположное!

align-content - распределение строк вдоль поперечной оси

align-content: stretch;растягиваются по поперечной оси (по умолчанию)

align-content: flex-start;в начале оси

align-content: flex-end;в конце оси

align-content: center;по центру

align-content: space-around;распределяются равномерно вдоль всей оси

align-content: space-between;равномерно (крайние элементы прибиваются к краям)

flex-flow - сокращенная запись flex-direction + flex-wrap

flex-flow: row nowrap;

свойства флекс-элементов

flex-grow - сколько свободного места займет элемент по сравнению с другими элементами (при наличии свободного пространства)
определяет насколько сильно будет увеличиваться флекс-элемент по отношению к другим флекс-элементам при распределении положительного свободного пространства

flex-grow: 0;по умолчанию - 0

flex-shrink - определяет на сколько сузится элемент по сравнению с другими элементами, чтобы все поместились в контейнере
определяет насколько сильно будет сжиматься флекс-элемент по отношению к другим флекс-элементам при распределении отрицательного свободного пространства

flex-shrink: 1;по умолчанию - 1

flex-basis - главный размер элемента (высота или ширина), т.е. размер вдоль главной оси

flex-basis: 100px;по умолчанию - auto

flex - сокращенная запись flex-grow + flex-shrink + flex-basis

flex: 0 1 auto;

order - порядок следования флекс-элементов

order: 0;по умолчанию - 0

align-self - распределение только этого элемента внутри строки вдоль поперечной оси

align-self: auto;те же значения, что и у align-items

Расчет flex-shrink и flex-grow

Свойство flex-shrink работает не совсем так же, как flex-grow... Если у последнего всё просто - положительное свободное пространство "раздаем" флекс-элементам в нужных пропорциях (в зависимости от значения flex-grow), то у flex-shrink это не работает (точнее сказать работает , но только при одинаковых flex-basis у элементов). То есть в случае flex-shrink, отрицательное свободное пространство "раздается" (то есть отнимается) не только в зависимости от значения flex-shrink, но и в зависимости от flex-basis элементов.

Так, размер элемента будет равен:


size = flex-basis + nfs * ( flex-basis * flex-shrink / ∑(flex-basis * flex-shrink) ),
	   где nfs - negative free space (отрицательное свободное пространство)
	       nfs = width_container - ∑flex-basis (значение получится отрицательным)

А для flex-grow формула будет почти идентична (только без учета flex-basis во второй части):


size = flex-basis + pfs * ( flex-grow / ∑flex-grow ),
	   где pfs - positive free space (положительное свободное пространство)
	       pfs = width_container - ∑flex-basis
